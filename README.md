# OnePageCRM API

The OpenAPI 3 specification for the [OnePageCRM](https://www.onepagecrm.com)
REST API — 165 operations covering contacts, companies, deals, actions,
notes, calls, and meetings.

This file is the source of truth for the API reference at
[developer.onepagecrm.com/api/reference/](https://developer.onepagecrm.com/api/reference/) —
every operation page there is generated from `swagger.yaml`, so a fix merged
here lands in the docs on the next site build.

## Use it

- **Browse the reference:** [developer.onepagecrm.com/api/reference/](https://developer.onepagecrm.com/api/reference/)
- **Generate a client:** point any [OpenAPI tool](https://openapi.tools) at [`swagger.yaml`](swagger.yaml)
- **Build with AI:** the docs ship agent-readable surfaces — `llms.txt`,
  per-page markdown mirrors, and a live MCP server. See
  [developer.onepagecrm.com/build-with-ai/](https://developer.onepagecrm.com/build-with-ai/).
